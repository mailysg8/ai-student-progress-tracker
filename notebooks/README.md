# Contents

This folder contains experiments that we ran throughout the project :

- [`best_subset_data.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/best_subset_data.ipynb) : Experiments with finding the best subset in our data
- [`bkt_baseline_model.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/bkt_baseline_model.ipynb) : Initial BKT experiments
- [`final_data.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/final_data.ipynb) : Experiments with building the final dataframe to use in the dashboard
- [`initial_eda.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/initial_eda.ipynb) : Initial EDA
- [`kc-agg_exploration.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/kc-agg_exploration.ipynb) : Explores different KC aggregation methods
- [`learnsphere_experiments.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/learnsphere_experiments.ipynb) : 
Explores using BKT on [LearnSphere](http://learnsphere.org) 
- [`pybkt-output.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/pybkt-output.ipynb) : Compares BKT performance with fine KC and modelling KC
- [`sample_exploration.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/sample_exploration.ipynb) : Experiments with creating samples with more students
- [`wk3_best_subset.ipynb`](https://github.com/mailysg8/ai-student-progress-tracker/blob/main/notebooks/wk3_best_subset.ipynb) : Further experiments to find the best subset.
